## 1. BRAND ESSENCE

**Настроение:** Нежность · Элегантность · Лёгкость  
**Эмоция:** «Это приглашение — как первый взгляд на свадьбу»  
**Аудитория:** Невесты 22–35 лет, ценят эстетику, хотят «красиво, но не пафосно»  
**Позиционирование:** Доступная премиальность — шаблоны от 2 000₽, которые выглядят на 10 000₽

## 3. TYPOGRAPHY

### Heading Font — Cormorant Garamond

```
font-family: 'Cormorant Garamond', Georgia, serif
```

**Почему:** Классический serif с элегантными пропорциями. Кириллица поддерживается. Выглядит премиально на любом размере. Как Vogue для свадеб.

| Уровень | Размер | Вес | Line-height | Letter-spacing | Применение |
|---------|--------|-----|-------------|----------------|------------|
| H1 (display) | 48px / 3rem | 600 | 1.1 | -0.02em | Hero-заголовок |
| H2 | 36px / 2.25rem | 600 | 1.15 | -0.01em | Секции |
| H3 | 28px / 1.75rem | 600 | 1.2 | 0 | Карточки, подсекции |
| H4 | 22px / 1.375rem | 500 | 1.25 | 0 | Мелкие заголовки |

### Body Font — Inter

```
font-family: 'Inter', -apple-system, BlinkMacSystemFont, sans-serif
```

**Почему:** Чистый, современный, идеальная кириллица. Читается на любом экране. Variable font — экономия загрузки.

| Уровень | Размер | Вес | Line-height | Letter-spacing | Применение |
|---------|--------|-----|-------------|----------------|------------|
| Body L | 18px / 1.125rem | 400 | 1.65 | 0 | Лид-абзацы |
| Body M | 16px / 1rem | 400 | 1.65 | 0 | Основной текст |
| Body S | 14px / 0.875rem | 400 | 1.55 | 0 | Подписи, карточки |
| Caption | 12px / 0.75rem | 400 | 1.4 | 0.02em | Метки, бейджи |
| Button | 15px / 0.9375rem | 500 | 1 | 0.04em | CTA кнопки |
| Overline | 11px / 0.6875rem | 600 | 1 | 0.12em | Категории, теги (UPPERCASE) |

### Decorative Font — Great Vibes

```
font-family: 'Great Vibes', cursive
```

**Почему:** Элегантный скрипт для акцентных слов. «Свадьба», имена молодожёнов, декоративные элементы. НЕ для body text.

| Уровень | Размер | Вес | Line-height | Применение |
|---------|--------|-----|-------------|------------|
| Script L | 56px / 3.5rem | 400 | 1 | Hero декоративное слово |
| Script M | 36px / 2.25rem | 400 | 1 | Имена, акценты |
| Script S | 24px / 1.5rem | 400 | 1 | Карточки, цитаты |

⚠️ **Great Vibes НЕ поддерживает кириллицу.** Использовать ТОЛЬКО для латинских слов/имен или латинской транслитерации декоративных элементов.

## 8. DO / DON'T — ВИЗУАЛЬНЫЕ ПРИМЕРЫ

### Цвет

**✅ DO:**
- Использовать Tiffany как доминантный акцент (CTA, линии, иконки)
- Ivory как основной фон — тепло и воздушно
- Gold ТОЛЬКО для акцентов: бейджи «Хит», иконки преимуществ, декоративные линии
- Нейтральный текст: `neutral-900` для заголовков, `neutral-700` для body

**❌ DON'T:**
- ❌ Tiffany как фон всей страницы → слишком интенсивно
- ❌ Gold для основного текста → контраст 3.2:1, не проходит WCAG AA
- ❌ Чёрный `#000000` → использовать `neutral-900` (#1A1816)
- ❌ Больше 3 акцентных цветов на одной странице

### Типографика

**✅ DO:**
- Cormorant Garamond для всех заголовков — единообразие и элегантность
- Inter для body — читаемость на мобильных
- Great Vibes ТОЛЬКО для 1-2 декоративных слов (латиница)
- Контраст размеров: H1 ≥ 2.5× body

**❌ DON'T:**
- ❌ Great Vibes для кириллицы → не поддерживается
- ❌ Больше 2 шрифтов на странице (не считая decorative)
- ❌ Body меньше 14px на мобильных
- ❌ Cormorant Garamond для body → serif плохо читается в мелком размере на экране

### Композиция

**✅ DO:**
- White space ≥ 20% площади макета → премиальность
- Контент по центру, max-width 960px → фокус на приглашении
- Z-паттерн чтения: лого → заголовок → CTA
- Один визуальный доминант на секцию

**❌ DON'T:**
- ❌ Растягивать контент на всю ширину экрана → максимум 960px
- ❌ Больше 1 CTA на viewport → паралич выбора
- ❌ Карточки без воздуха между ними → минимум gap 16px
- ❌ Текст поверх фото без overlay gradient → нечитаемость

### Карточки каталога

**✅ DO:**
- Соотношение сторон 3:4 (портретное) → как приглашение
- Тень `shadow-sm` → `shadow-md` при hover
- Border-radius 12px → мягко, но не «игрушечно»
- Бейдж «Хит» / «Новинка» — gold-500 на gold-100

**❌ DON'T:**
- ❌ Тень + бордер одновременно → визуальный шум
- ❌ Больше 2 строк в названии шаблона
- ❌ Цена без знака ₽
- ❌ Hover-эффект, меняющий layout (сдвиг контента)
