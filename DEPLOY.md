# 🚀 Деплой сайта gromovaasya.ru

> Пошаговая инструкция запуска сайта свадебных приглашений  
> **Время:** 15–20 минут  
> **Стоимость:** 0₽ (бесплатный хостинг)

---

## Шаг 1: Создать GitHub репозиторий (5 мин)

1. Зайти на [github.com](https://github.com) → **Sign up** (если нет аккаунта)
2. Нажать **New repository**
3. Название: `gromovaasya-wedding-site`
4. Поставить галочку **Public**
5. Нажать **Create repository**

### Загрузить файлы

В терминале (или через GitHub Desktop):

```bash
cd ~/.hermes/projects/сайты/свадебные-сайты/gromovaasya-wedding-site

# Привязать к репозиторию (замените USERNAME на ваш логин)
git remote add origin https://github.com/USERNAME/gromovaasya-wedding-site.git
git branch -M main
git push -u origin main
```

Или через веб: **Upload files** → перетащить все файлы → **Commit**.

---

## Шаг 2: Включить GitHub Pages (3 мин)

1. В репозитории → **Settings** → **Pages** (в меню слева)
2. **Source** → выбрать **GitHub Actions**
3. Готово! Через 2–3 минуты сайт будет доступен по адресу:
   ```
   https://USERNAME.github.io/gromovaasya-wedding-site/
   ```

> 💡 **Кастомный домен:** Если хотите `gromovaasya.ru`, создайте файл `CNAME` в корне репозитория с текстом `gromovaasya.ru` и настройте DNS у регистратора.

---

## Шаг 3: Настроить Formspree (5 мин)

Formspree отправляет заявки из формы на вашу почту.

1. Зайти на [formspree.io](https://formspree.io) → **Sign up**
2. Создать новый проект → **New Form**
3. Название: «gromovaasya заявки»
4. Email: ваш email для получения заявок
5. Скопировать ID формы (например: `xmqbzqwe`)
6. В файле `index.html` заменить `[FORMSPREE_ID]` на ваш ID:
   ```html
   <!-- Было: -->
   action="https://formspree.io/f/[FORMSPREE_ID]"
   <!-- Стало: -->
   action="https://formspree.io/f/xmqbzqwe"
   ```
7. Закоммитить и пушнуть:
   ```bash
   git add index.html && git commit -m "Настроить Formspree" && git push
   ```

**Проверить:** Откройте сайт → заполните форму → проверьте почту ✅

## Шаг 4: Подключить Яндекс.Метрику (5 мин)

1. Зайти на [metrika.yandex.ru](https://metrika.yandex.ru)
2. **Добавить счётчик** → название «gromovaasya»
3. Адрес сайта: ваш URL из Шага 2
4. Включить: **Вебвизор**, **Карта скроллинга**, **Карта кликов**
5. Скопировать номер счётчика (например: `123456789`)
6. В файле `index.html` найти блок `<!-- Яндекс.Метрика -->`:
   - Раскомментировать (удалить `<!--` и `-->`)
   - Заменить `[METRIKA_ID]` на ваш номер (2 раза)
7. Закоммитить и пушнуть

**Настроить цели** в Метрике:
| Цель | Тип | Идентификатор |
|------|-----|---------------|
| Отправка формы | JavaScript-событие | `order_submit` |
| Клик по фильтру | JavaScript-событие | `filter_use` |
| Открытие модалки | JavaScript-событие | `template_view` |
| Клик на Telegram | JavaScript-событие | `contact_tg` |
